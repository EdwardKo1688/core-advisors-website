📋 Phase 2 功能設計

🆕
12.6.1 案件建立

從診斷紀錄或留資名單一鍵建立顧問案件，自動帶入診斷結果與風險標記。

📊
12.6.2 漏斗追蹤

診斷完成 → 留資 → 預約確認 → 服務進行中 → 完案，全程可視化。

📝
12.6.3 案件備注

每個案件可記錄顧問備注、行動清單完成度、下次跟進時間。

📈
12.6.4 轉換率分析

診斷→留資→預約→成案轉換漏斗，協助優化諮詢轉化效率。

報告匯出 Phase 2

將診斷結果匯出為格式化 PDF 或 Excel 報告，供顧問與客戶使用。

📊 Phase 2 功能設計

📄
12.7.1 PDF 診斷報告

含公司名稱、成熟度等級、雷達圖、Top-3 風險、Top-5 行動建議的完整診斷報告。

📊
12.7.2 Excel 資料匯出

批次匯出所有 IC 診斷紀錄含各模型分數明細，支援後續資料分析。

📈
12.7.3 管理儀表板報告

週期性（月/季）診斷統計報告，含角色分佈、平均分數、常見風險趨勢。

📧
12.7.4 自動寄送

診斷完成後自動 Email 報告摘要給填答者（需 GAS SendMail 整合）。

PDF 報告結構設計

第 1 頁：封面（公司名、日期、角色、等級徽章）
第 2 頁：成熟度概覽（雷達圖 + 五模型分數條）
第 3 頁：Top-3 風險分析（詳細說明 + 觸發條件）
第 4 頁：Top-5 行動建議（P1/P2/P3 + 建議時程）
第 5 頁：附錄（評分方法說明、版本資訊）
