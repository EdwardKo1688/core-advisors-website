## Why

台灣人力仲介產業正從「媒合導向」進入「治理導向」競爭階段。官方評鑑制度（附表一/二/三）已明確要求仲介機構具備契約治理、建檔管制、資訊安全、教育訓練、滿意度改善、多語服務、申訴處理與經營穩定等八大能力。然而現有系統（如 SandboxTech Hello 移工管理系統）僅聚焦於訪視表管理與基礎雇主/移工 CRUD，無法支撐評鑑準備、法遵控制、服務品質證據鏈與經營決策。產業最大的問題不是沒做事，而是做過的事無法形成可追蹤、可回溯、可證明的管理證據。現在正是打造新一代「評鑑驅動 + AI 輔助」一體化營運平台的時機。

## What Changes

- 建立完整的**評鑑管理中台**：將官方三份附表（外國人仲介/未辦理案件/本國人仲介）的評鑑指標數位化，自動追蹤缺件、產生佐證清單
- 建立**案件全生命週期管理**：從雇主委任→移工媒合→入境→服務期間→續聘/轉換→離境，每個節點產生結構化紀錄與證據
- 建立**電子卷宗與契約治理**系統：契約、訪視紀錄、收費明細、申訴紀錄統一歸檔，可即時抽查
- 建立**訪視服務運營引擎**：服務日曆、任務派工、追蹤時限、主管覆核、多語訪視表，升級現有訪視管理
- 建立**法遵風控引擎**：收費規則檢核、流程風險提示、法規知識庫、違規警示
- 建立**異常事件管理**工作流：事件分級、SLA、升級規則、資源名冊、案例沉澱
- 建立**滿意度調查與分析**系統：多語問卷、結果連結續約/客訴/流失風險
- 建立**教育訓練與知識傳承**模組：年度訓練紀錄、新人交接清單、知識庫
- 建立 **6 個 AI Copilot 模組**：評鑑 Copilot、法遵風控 Copilot、案件摘要與交接助手、異常事件助手、滿意度分析、經營決策 Copilot
- 建立**經營儀表板**：評鑑落點預測、風險熱點、分公司績效、客戶流失預警
- 建立**多角色協同**機制：總部/分公司/業務/翻譯/行政/主管/稽核專員，共用案件時間軸
- 提供 **PWA 行動端**：取代現有 Android-only APP，支援 iOS/Android 雙平台

## Capabilities

### New Capabilities
- `assessment-engine`: 評鑑指標管理引擎——將附表一/二/三評鑑指標數位化，自動追蹤完成度、缺件檢查、佐證清單產生、評鑑落點預測
- `case-lifecycle`: 案件全生命週期管理——從雇主委任到離境的完整案件時間軸，涵蓋契約、媒合、入境、服務、續聘、轉換、離境各階段
- `document-vault`: 電子卷宗與契約治理——統一文件歸檔、版本管理、缺件提醒、Completeness Score、可即時抽查的證據鏈
- `visit-operations`: 訪視服務運營引擎——服務日曆、任務派工、多語訪視表、審核流程、追蹤時限、訪視提醒與追蹤，升級現有訪視管理功能
- `compliance-engine`: 法遵風控引擎——收費規則引擎、退費檢核、續聘/轉換流程規則、法規知識庫、違規風險警示
- `incident-management`: 異常事件管理——事件工單、分級分類、SLA 管控、升級規則、資源名冊、通報機制、案例知識庫
- `satisfaction-survey`: 滿意度調查與分析——多語問卷建置、回收追蹤、結果與續約/客訴/流失風險連結、改善閉環
- `training-knowledge`: 教育訓練與知識傳承——年度訓練計畫、出勤紀錄、新人交接清單、知識庫、回訓機制
- `ai-copilots`: AI 輔助助手模組群——評鑑 Copilot、法遵 Copilot、案件摘要助手、異常事件助手、滿意度分析、經營決策 Copilot
- `executive-dashboard`: 經營決策儀表板——評鑑落點預測、風險排名、分公司績效、客戶流失預警、週報自動產生
- `rbac-multitenancy`: 角色權限與多租戶——7 種角色權限、總部/分公司多層級架構、資料隔離
- `pwa-mobile`: PWA 行動應用——取代 Android-only APP，支援離線訪視表填寫、簽名、拍照、GPS 定位

## Impact

- **前端**：全新 React SPA 開發，參考現有雛形網站設計風格（深色導航、Teal 主色調），需支援 12+ 個主要頁面模組
- **後端**：Node.js/Express 或 Python/FastAPI RESTful API，需設計 20+ 個核心資料模型
- **資料庫**：PostgreSQL，需支援多租戶資料隔離、全文搜尋、JSON 欄位（彈性欄位）
- **AI 整合**：Claude API 整合，需設計 prompt 工程、向量資料庫（法規知識庫）、RAG pipeline
- **檔案儲存**：S3/MinIO 物件儲存，支援契約、訪視照片、簽名檔等文件管理
- **行動端**：PWA with Service Worker，支援離線填表與同步
- **外部整合**：Google Calendar API、Email/SMS 通知、PDF 匯出引擎
- **部署**：Docker 容器化，AWS/GCP 雲端部署，CI/CD pipeline
- **安全**：JWT 認證、RBAC 授權、個資加密、稽核日誌
