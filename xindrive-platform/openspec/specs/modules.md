## Module: AUTH — 用戶與權限管理

### AUTH-001: 登入頁

- 品牌級視覺（深色漸層背景 + 白色登入卡片）
- Email + 密碼登入表單
- Demo 快速角色切換（3 個角色按鈕：林總監/陳業務/張教練）
- 登入後根據角色跳轉 `#/dashboard`
- Token 存入 localStorage
- 登入失敗顯示 Toast 錯誤訊息
- 底部品牌 Tagline：三支柱圖示 + 「不忘初心，AI倍增」

### AUTH-002: 角色權限 (RBAC)

| 角色 | 代碼 | 說明 | Dashboard 視角 |
|------|------|------|----------------|
| 管理員 | admin | 完整權限 | 組織全局 |
| 主管 | manager | 管理 + 查看全部 | 團隊 KPI + Pipeline |
| 講師/教練 | trainer | 培訓管理 + 陪跑 | 我的學員 + 課程 |
| 業務 | sales | 學習 + CRM + 業績 | 我的學習 + 我的業績 |

### AUTH-003: 角色切換（Demo 專用）

- Top Bar 用戶下拉選單包含「切換角色」
- 切換後重新載入 Dashboard，顯示對應視角
- 無需重新登入

## Module: DASHBOARD — 管理儀表板

### DASH-001: 角色差異化首頁

根據當前角色顯示不同的 Dashboard 佈局：

#### Manager 視角
- 歡迎訊息 + 公司名 + 日期
- 4 張 KPI Stat Card：本月營收 / 活躍商機 / 學習完成率 / 陪跑進行
- 營收趨勢折線圖（6 個月）
- Pipeline 分佈長條圖
- 團隊技能雷達圖（8 維度）
- 最近動態時間軸

#### Sales 視角
- 歡迎訊息 + 個人化
- 我的學習進度卡片（當前課程 + 進度條 + 下一步）
- 我的陪跑狀態（教練、下次 Session、待辦行動項目）
- 我的業績卡片（本月目標 vs 實際 + 達成率）
- 我的 Pipeline 摘要（商機數、加權金額）
- 我的待辦清單

#### Trainer 視角
- 歡迎訊息 + 個人化
- 學員總覽卡片（帶教人數、平均進度）
- 我的陪跑計畫（進行中 Session、排程）
- 課程數據（我管理的課程、完課率、評分）
- 學員能力成長曲線圖

### DASH-002: KPI Stat Card 元件

```
┌────────────────┐
│ 📈  本月營收     │
│                │
│   NT$23.5M     │  ← 大數字（粗體）
│   ▲ 12% MoM    │  ← 趨勢指標（綠漲紅跌）
└────────────────┘
```

- 4 種變體：default / learn / coach / boost（左邊色條）
- 趨勢箭頭：▲ 綠色（正成長）/ ▼ 紅色（負成長）/ — 灰色（持平）

### DASH-003: 最近動態

- 時間軸式列表
- 每筆動態包含：圖示 + 描述 + 時間
- 動態來源：學習進度、商機變更、陪跑 Session、系統通知
- 最多顯示 10 筆

## Module: LEARN — 驅・知 培訓管理

### LEARN-001: 課程目錄

- 路由：`#/learn`
- 頂部篩選列：分類（銷售/採購/供應鏈/管理）| 難度（初階/中階/高階）| 排序
- 卡片網格顯示（Desktop 3 欄 / Tablet 2 欄 / Mobile 1 欄）
- 課程卡片內容：
  - 封面圖（或分類色塊佔位）
  - 課程標題
  - 分類 Badge + 難度 Badge + 時數 + 模組數
  - 課程簡述（2 行截斷）
  - 進度條（已選課者）
  - CTA 按鈕：「開始學習」/「繼續學習」/「查看詳情」
- 搜尋：標題 + 描述即時篩選

### LEARN-002: 課程詳情/學習介面

- 路由：`#/learn/course/:id`
- 左側面板：模組/課堂目錄（手風琴展開收合）
  - 每堂顯示：標題、類型圖示、時長、完成狀態勾選
- 右側主區：當前課堂內容
  - 支援類型：文字說明、影片（placeholder）、測驗、練習
- 頂部：課程標題 + 整體進度條
- 底部：上一堂 / 下一堂 導航
- 完成課堂後自動更新進度

### LEARN-003: 我的學習進度

- 路由：`#/learn/progress`
- 個人學習儀表板
- 已選課程列表 + 各課程完成百分比
- 總學習時數統計
- 技能雷達圖（根據完成課程的分類計算）
- 學習日曆熱力圖（可選）

### LEARN-004: 課程管理（Admin/Trainer）

- 路由：`#/learn/manage`
- 權限：Admin, Manager, Trainer
- 課程 CRUD 表格
- 新增/編輯課程 Modal：
  - 基本資訊：標題、分類、難度、描述
  - 模組管理：新增/刪除/排序模組
  - 課堂管理：每模組下新增/刪除/排序課堂
- 發佈/下架 Toggle
- 學員名單查看

## Module: COACH — 驅・伴 陪跑管理

### COACH-001: 陪跑中心

- 路由：`#/coach`
- 我的陪跑計畫卡片列表
- 每張計畫卡片：
  - 教練/學員配對
  - 方法論 Badge（GROW/MEDDIC/SPIN/CHALLENGER）
  - 進度：已完成 Session 數 / 總計劃
  - 目標達成狀態
  - 下一次 Session 日期
  - 狀態：進行中 / 已完成 / 暫停
- Manager 可看團隊所有計畫
- Sales 只看自己的計畫

### COACH-002: AI 教練對話介面

- 路由：`#/coach/ai`
- 對話式介面（類 ChatGPT，品牌化設計）
- 頂部資訊列：方法論、場景、Session 編號
- 對話氣泡：
  - 教練（左側）：AI 回應，帶 🤖 圖示
  - 用戶（右側）：用戶輸入，帶 👤 圖示
- 底部輸入框 + 送出按鈕
- Demo 模式：預載 2-3 輪對話作為範例
- 場景選擇：客戶拜訪模擬 / 談判練習 / QBR 簡報練習
- **Note**：Demo 版為靜態對話展示，非真實 AI 互動

### COACH-003: Session 管理

- 路由：`#/coach/session/:id`
- Session 記錄表單：
  - Session 類型（1on1/roleplay/review/assessment）
  - 會議筆記（文字區塊）
  - 行動項目列表（可勾選完成）
  - 回饋與評分
  - 下一次 Session 排程
- Session 歷史列表

### COACH-004: 能力評估

- 路由：`#/coach/assessment`
- 8 維度技能雷達圖：
  1. 產品知識
  2. 客戶開發
  3. 需求挖掘
  4. 方案提案
  5. 談判技巧
  6. 關係管理
  7. 市場分析
  8. 數位工具
- 三層雷達疊加：當前 / 3 個月前 / 目標值
- 各維度分數 1-10
- 建議學習路徑（根據弱項推薦課程）
- 評估歷史時間軸

## Module: BOOST — 驅・效 業績管理

### BOOST-001: 業績總覽

- 路由：`#/boost`
- 核心 KPI 卡片：
  - 本月/季/年 營收 vs 目標
  - 商機轉換率
  - 新客戶數
  - 平均成交週期
- 營收趨勢圖（折線）
- 業績排行榜（團隊成員排名）
- 切換：個人 / 團隊 / 月度 / 季度

### BOOST-002: 客戶管理

- 路由：`#/boost/customers`
- 客戶列表（表格）
- 欄位：客戶名稱、產業、分級(Tier)、負責人、狀態、營收
- 篩選：分級 / 狀態 / 負責人
- 搜尋：客戶名稱
- 點擊展開客戶詳情：
  - 基本資料
  - 聯絡人列表
  - 相關商機
  - 活動歷史
- 新增/編輯客戶 Modal

### BOOST-003: 商機 Pipeline

- 路由：`#/boost/pipeline`
- **Kanban 看板**（核心互動元件）
- 5 個階段欄位：
  - Lead → Qualified → Proposal → Negotiation → Closed Won
- 每欄顯示：商機數 + 加權金額
- 商機卡片：
  - 商機名稱
  - 客戶名稱
  - 預估金額
  - 成交機率（色彩編碼）
  - 預計成交日
  - 負責人 Avatar
- 點擊商機卡片展開詳情 Drawer：
  - 完整商機資訊
  - MEDDIC 評分卡
  - 活動記錄時間軸
  - 下一步行動
- 拖曳移動商機（更新階段）
- Lost 區域（已流失商機）

### BOOST-004: 業績目標

- 路由：`#/boost/targets`
- 目標 vs 實績對比圖（長條圖）
- 達成率進度環
- 個人/團隊切換
- 月度/季度/年度切換
- 目標設定（Admin/Manager）：
  - 選擇人員
  - 設定類型（營收/Design-in/新客戶/毛利率）
  - 設定目標值
  - 設定期間

## Module: SYSTEM — 共用功能

### SYS-001: 通知中心

- Top Bar 鈴鐺圖示 + 未讀數量 Badge
- 下拉通知列表
- 通知類型：學習提醒、Session 排程、商機更新、系統公告
- 標記已讀

### SYS-002: Toast 通知

- 右上角浮現
- 類型：success / error / warning / info
- 3 秒自動消失
- 可手動關閉

### SYS-003: Modal / Drawer

- Modal：居中彈窗（表單、確認對話框）
- Drawer：右側滑入面板（詳情查看）
- 背景遮罩 + ESC 關閉
