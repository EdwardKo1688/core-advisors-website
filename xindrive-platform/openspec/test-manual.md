**網址**：http://localhost:8081/#/login

| Trainer | 張教練 | 張教練 | chang@sinda.com |

### 操作步驟

1. **Demo 角色登入（推薦）**
   - 點擊下方三個角色卡片之一
   - 預期：顯示 Toast「歡迎回來，[姓名]」→ 跳轉至儀表板

2. **Email 登入**
   - 輸入 Email（如 `lin@sinda.com`）+ 任意密碼
   - 點擊「登入」或按 Enter
   - 預期：成功跳轉至儀表板

3. **錯誤測試**
   - 空白 Email → 顯示「請輸入電子郵件」
   - 不存在的 Email → 顯示「找不到此帳號，請使用 Demo 角色登入」
   - 錯誤訊息 3 秒後自動消失

## 3. 驅知 LEARN

### 3A. 課程目錄

**網址**：http://localhost:8081/#/learn

**操作步驟**：

1. 確認顯示 4 門 Demo 課程：
   - MEDDIC 銷售方法論（銷售/高階）
   - IC 供應鏈基礎（供應鏈/中階）
   - 採購談判策略（採購/高階）
   - 營運管理實務（經營管理/中階）

2. **篩選測試**
   - 分類下拉選「銷售」→ 只顯示 MEDDIC
   - 難度下拉選「中階」→ 顯示供應鏈 + 營運管理
   - 搜尋輸入「MEDDIC」→ 只顯示 MEDDIC
   - 清空篩選 → 恢復全部 4 門

3. **課程卡片**
   - 每張卡片顯示：Emoji 圖示、標題、分類/難度/時數 Badge、描述、進度條
   - 按鈕：「開始學習」/「繼續學習」/「複習課程」

### 3B. 課程詳情

**網址**：http://localhost:8081/#/learn/course/1（以 MEDDIC 為例）

**操作步驟**：

1. 從課程目錄點擊任一課程
2. 確認左側顯示模組手風琴（可展開/收合）
3. 點擊不同課堂 → 右側內容區更新
4. 點擊「完成並繼續 →」→ 課堂標記 ✓ + 自動跳至下一堂
5. 點擊「← 上一堂」→ 返回前一堂
6. 確認頂部進度條隨完成數量更新

### 3C. 我的學習進度

**網址**：http://localhost:8081/#/learn/progress

**操作步驟**：

1. 點擊 Sub-Tab「📊 我的學習」
2. 左側：已選修課程列表 + 進度條
3. 右側：技能分佈雷達圖（8 維度、目前 vs 目標兩條線）
4. 點擊課程名稱 → 跳轉至課程詳情

### 3D. 課程管理（僅 Trainer/Manager）

**網址**：http://localhost:8081/#/learn/manage

**操作步驟**：

1. 以張教練或林總監登入
2. 點擊 Sub-Tab「⚙ 課程管理」
3. 確認表格顯示所有課程（名稱/分類/難度/時數/狀態）
4. 點擊欄位標題 → 排序切換
5. 點擊「+ 新增課程」→ Modal 開啟
   - 填寫：課程名稱、分類、難度、時數、描述
   - 點擊「建立」→ Toast「課程已建立」
6. 點擊表格中某一列 → 編輯 Modal 開啟
   - 修改內容 → 點擊「更新」→ Toast「課程已更新」

> ⚠️ 以陳業務登入時，「⚙ 課程管理」Tab 不可見

## 4. 驅伴 COACH

### 4A. 陪跑計畫

**網址**：http://localhost:8081/#/coach

**操作步驟**：

1. 確認顯示陪跑計畫卡片
2. 每張卡片：計畫標題 + 方法論 Badge + 教練→學員配對 + Session 數 + 狀態
3. 點擊卡片 → 跳轉至 Session 紀錄頁

**角色差異**：
- Sales：僅看到自己參與的計畫
- Trainer：看到自己擔任教練/學員的計畫
- Manager：看到所有計畫

### 4B. AI 教練

**網址**：http://localhost:8081/#/coach/ai

**操作步驟**：

1. 點擊 Sub-Tab「🤖 AI 教練」
2. 確認顯示 Demo 對話（MEDDIC 客戶拜訪模擬）
3. 在輸入框輸入文字 → 點擊「送出」或按 Enter
4. 預期：用戶訊息出現在右側 → 約 1 秒後 AI 回應出現在左側
5. AI 回應為 Demo 模式固定回饋（非即時 AI）

### 4C. Session 紀錄

**網址**：http://localhost:8081/#/coach/session/1

**操作步驟**：

1. 從陪跑計畫卡片點擊進入
2. 確認 Session 列表（編號、主題、日期、時長、狀態）
3. 點擊任一 Session → 展開詳情：
   - 📝 會議筆記
   - ✅ 行動項目（勾選清單）
   - 💬 回饋（優勢 + 待改善）
4. 點擊「← 返回」→ 回到陪跑計畫列表

### 4D. 能力評估

**網址**：http://localhost:8081/#/coach/assessment

**操作步驟**：

1. 點擊 Sub-Tab「📈 能力評估」
2. 左側：技能雷達圖（目前/3 個月前/目標 三條線）
3. 右側：各維度評分列表
   - 8 個維度：指標量化、經濟買家、決策標準、決策流程、痛點識別、Champion、簡報能力、談判技巧
   - 每項顯示分數條 + 趨勢指標（↑ 綠色表示進步）
4. 底部：綜合評分（如 72/80）+ 評估摘要

## 5. 驅效 BOOST

### 5A. 業績總覽

**網址**：http://localhost:8081/#/boost

**操作步驟**：

1. 確認 4 個 KPI 卡片：
   - Pipeline 總額 / 加權預測 / 本季成交 / 勝率
2. 圖表：各階段 Pipeline 長條圖 + 月營收趨勢折線圖
3. 業績排行榜（依實際營收排序）
4. 重點案件（Top 5 金額 ≥NT$1M 的活躍商機）
5. 點擊重點案件 → 跳轉至 Pipeline 頁

### 5B. 客戶管理

**網址**：http://localhost:8081/#/boost/customers

**操作步驟**：

1. 確認表格顯示 7 家客戶
2. **搜尋測試**：輸入客戶名稱 → 即時篩選
3. **層級篩選**：選「Tier 1」→ 只顯示 Tier 1 客戶
4. 點擊表格列 → 右側抽屜開啟，顯示：
   - 基本資訊（產業/層級/區域/年營收）
   - 聯絡人列表（姓名/職稱/Email）
   - 商機列表（名稱/金額/階段）
5. 點擊抽屜外部或關閉按鈕 → 關閉

### 5C. Pipeline 看板

**網址**：http://localhost:8081/#/boost/pipeline

**操作步驟**：

1. 確認 4 欄看板：線索 / 確認 / 提案 / 談判
2. 每張卡片：商機名稱 + 客戶 + 金額 + 機率 + 預計結案日
3. **拖拽測試**：
   - 拖動一張卡片至另一欄
   - 預期：階段更新 + 機率自動調整（Lead 10% / Qualified 30% / Proposal 50% / Negotiation 70%）
   - 看板即時重新渲染
4. 點擊卡片 → 右側抽屜顯示：
   - 商機詳情（客戶/負責人/金額/機率/預計結案）
   - 描述 + 備註
   - 活動記錄（🤝會議/📧郵件/📞電話/📝筆記）
5. 底部摘要：Pipeline 總額 / 加權預測 / 商機數

### 5D. 業績目標

**網址**：http://localhost:8081/#/boost/targets

**操作步驟**：

1. **團隊目標卡片**：
   - 左側：圓形進度環（達成率 %）
   - 右側：目標 NT$80M / 實際 NT$62M / 差距 / 剩餘天數
2. **個人目標卡片**：
   - 長條圖（每人實際營收）
   - 個人列表：姓名 + 達成率 Badge + 進度條 + 實際/目標金額
   - Badge 顏色：≥80% 綠 / ≥60% 橘 / <60% 紅
